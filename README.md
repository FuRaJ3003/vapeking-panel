# VapeKing Panel (vapeking-panel) 
A panel for managing employee data and a local product warehouse for the vapeking store chain.             
The panel is based on Python / Django and GraphQL.


<i>Basic Req:</i> 
> Django 3.2 /
> Python 3.9 /
> Graphene-Django 2.1.8 /
